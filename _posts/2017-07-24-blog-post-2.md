## 560. Subarray Sum Equals K
<pre>
class Solution(object):
    
    def subarraySum(self, nums, k):
        """
        :type nums: List[int]
        :type k: int
        :rtype: int
        """
        dict = {0:1}
        res = sum = 0
        for num in nums:
            sum += num
            if sum-k in dict:
                res += dict[sum-k]
            if sum in dict:
                dict[sum] += 1
            else:
                dict[sum] = 1
        return res
</pre>
Time Complexity: O(n), Space Complexity: O(n).
